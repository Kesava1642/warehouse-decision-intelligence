# BCP Service — One-Page Summary

| Item | Detail |
|------|--------|
| **Service name** | Business Continuity Plan (BCP) — Lean Template |
| **What it is** | A short (2–3 page), executable BCP tailored to the client’s operation. Role-based, incident-ready. |
| **What client gets** | Word/PDF BCP with: Document Control; Purpose & Activation (L1/L2/L3); Governance roles; Critical Activities & MTD/RTO; 6–8 Key Scenarios; First 60-min checklist; Communications Plan; Return to Normal; Review & Testing. Plus one revision. |
| **Intake** | 10–12 questions (locations, departments, critical activities, MTD/RTO, key systems, comms, governance roles, top scenarios). |
| **Build time** | 60–90 min (see BCP_Build_SOP.md). |
| **Deliverables** | Final BCP (Word/PDF); one revision round. |
| **Internal docs** | BCP_Lean_Template.md, BCP_Intake_Form.md, BCP_Build_SOP.md. |
